# Financial Reporting Dashboard | Power BI

An interactive 6-page Power BI dashboard built to give end-to-end 
visibility into company financial performance — from Income Statement 
through Revenue Insights — using DAX-driven reconciliation across 
multiple financial and transactional data sources.

## 🔧 What this dashboard does
- Consolidates Sales Orders, Customer, Regional, Product, and 
  Company Expense data into a single reporting model
- Custom DAX measures reconcile figures across statements and 
  calculate key financial metrics
- Six interconnected report pages accessible via a central 
  navigation menu
